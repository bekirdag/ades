06 — Open source tool assignments
Ingestion
Trafilatura
Best-in-class HTML→text extraction. Apache 2.0. pip install trafilatura
Ingestion
pymupdf · ftfy · langdetect
PDF extraction, unicode repair, language detection
NLP base
spaCy
Tokenize, POS, dep parse, sentence split. en_core_web_trf for quality.
NLP base
coreferee
Coreference resolution as spaCy pipeline component
NER
GLiNER
Zero-shot, any entity type, outperforms ChatGPT on NER benchmarks. NAACL 2024.
Patterns
FlashText + spaCy EntityRuler
ISINs, tickers, money amounts, phone numbers, URLs. ~4000x faster than regex at scale.
Topic classification
SetFit
Few-shot: 8 examples per class, 5x faster inference than NLI pipeline. HuggingFace.
Topic classification
facebook/bart-large-mnli
Zero-shot fallback. No training required, pass candidate labels at inference time.
Entity linking
REL (Radboud Entity Linker)
Best quality self-hosted Wikipedia/Wikidata linker. REST-API deployable.
Entity linking
OpenTapioca
Lightweight Wikidata linker. Real-time sync with Wikidata. Apache 2.0.
Vector search
FAISS / Qdrant
ANN retrieval over entity description embeddings for candidate generation
Relation extraction
REBEL (Babelscape)
Seq2seq → (subject, relation, object) triples. ~220 Wikidata relation types.
Relation extraction
GLiNER multitask
Joint NER + relation extraction in one forward pass. Knowledgator model.
Storage
Postgres + OpenSearch + Redis
Entity records, full-text search + faceting, async queue + caching
Knowledge base
Wikidata dump → SQLite / Postgres
Entity records, aliases, types, descriptions. Seeded via wikid spaCy project.
API framework
FastAPI + Celery + Dramatiq
Sync + async endpoints. Worker pool for batch tagging jobs.
